# Giggle Examples

This directory contains example programs demonstrating the usage of the giggle audio analysis library. These examples are inspired by the [librosa example gallery](https://librosa.org/doc/latest/auto_examples/index.html).

## Running Examples

```bash
# Run a specific example
cargo run --example basic_features

# Run with release mode for better performance
cargo run --release --example chroma_features
```

## Available Examples

### Basic Usage

- **`basic_features`** - Basic feature extraction (MFCC, mel spectrogram, chroma)
- **`spectral_analysis`** - STFT, mel spectrogram, and spectral feature analysis
- **`music_synthesis`** - Signal generation (tones, chirps, clicks, chords)

### Feature Extraction

- **`chroma_features`** - Chroma feature extraction (STFT, CQT, CENS)
- **`mfcc_extraction`** - MFCC extraction
- **`spectral_contrast`** - Spectral contrast analysis

### Effects and Separation

- **`hpss`** - Harmonic-percussive source separation
- **`vocal_separation`** - Vocal/instrumentation separation using soft masking (REPET-SIM inspired)
- **`trim_split`** - Audio trimming and splitting
- **`time_pitch_effects`** - Time stretching and pitch shifting

### Rhythm and Timing

- **`beat_tracking`** - Tempo and beat tracking
- **`onset_detection`** - Onset detection
- **`superflux_onset`** - Superflux onset detection (vibrato-robust, Boeck & Widmer 2013)

### Pitch and Harmony

- **`pitch_tracking`** - Pitch tracking with YIN algorithm
- **`harmonic_analysis`** - Harmonic spectrum extraction and analysis (f0 harmonics, salience)

### Sequence Analysis

- **`dtw_alignment`** - Dynamic Time Warping for sequence alignment
- **`music_sync`** - Music synchronization with DTW for tempo-varying alignment
- **`viterbi_decoding`** - Viterbi decoding for temporal smoothing

## Example Data

Examples use synthetic signals generated by the library's signal generators (`tone`, `chirp`, `clicks`).
